## [Unreleased]

### Added

- CI/CD pipeline with Blue-Green deployment strategy
- Conventional commits validation
- Branch naming conventions enforcement
- Automated changelog generation
- Dependabot for dependency updates
- Git hooks with Husky for local validation

## [1.0.0] - 2026-02-04

### Added

- Initial release of SHIFT - The Board Game
- Multiplayer real-time gameplay with Socket.io
- Dynamic rules system with triggers, conditions, and effects
- 10+ predefined rule packs (Vanilla, Classic, Challenge, etc.)
- Interactive tutorial system with 11 steps
- Friend system and in-game chat
- Bot AI with 3 difficulty levels
- Game history and statistics
- PWA support for offline play
